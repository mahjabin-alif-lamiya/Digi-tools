# DigiTools

DigiTools is a modern digital tools platform built with React.js.  
Users can browse different tools, add them to the cart, and switch between product and cart views with a smooth user experience.


---

## 📌 Project Overview

This project focuses on creating a clean and interactive frontend experience using React and modern UI technologies.  
It demonstrates component-based architecture, dynamic state management, and responsive design principles.

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6)
- React Toastify

---

## ✨ Features

- Browse digital tools dynamically
- Add and remove products from cart
- Product and cart toggle system
- Interactive toast notifications
- Responsive modern UI design
- Smooth user experience
